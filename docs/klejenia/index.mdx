# 🏗️ System klejenia i konfiguracja głowic

Instrukcja identyfikacji metod aplikacji oraz lokalizacji głowic na sterowaniu maszyny.

## 🛠️ Metody aplikacji (Logika podawania)

- **Continuum:** Klejenie odbywa się w sposób ciągły (nieprzerwany).
- **Count:** Klejenie taktowane (podawanie impulsowe).
- **Allegro:** Specjalistyczna aplikacja przeznaczona do klejenia nitek lycry.
